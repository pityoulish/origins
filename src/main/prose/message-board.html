Message Board

The Message Board is at the center of the programming exercises. Its basic functionality remains the same, regardless of the communication technology employed between clients and the server.

The Message Board resides on the server. It contains a limited number of messages. Clients can read these messages in batches. Clients can add new messages, one at a time. Adding a message requires a valid ticket. The server may generate system messages as well. Old messages eventually drop off the board.
