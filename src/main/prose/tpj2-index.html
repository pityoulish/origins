Tutorial: Prepare for Java Exercises

...

objectives:

  • set up the development environment
  • learn how to navigate an unfamiliar codebase
  • practice the steps for fixing a program
  • avoid common pitfalls

If you're an experienced Java developer, all of this will be familiar to you. But many students in my classes are not, so this tutorial establishes a baseline of installed tools and practiced skills for the classroom exercises.

Setup

  • Browser
  • Java SDK
  • Editor
  • ...or your favorite IDE for Java development
  • useful documentation, as link or copy (standard JavaDocs)

Download

tutorial-faulty.jar
The program to test and fix. Usage instructions:
  java -jar tutorial-faulty.jar
Sources are in the jar, docs here.
  • download the program, change the filename
  • run the program
  • prepare for fixing it, with or without IDE

IDE

  • create a project from the JAR, not from extracted sources
  • build your own JAR
  • run your own JAR
  • Is your JAR executable? Are the properties files included?

Command Line

  • extract the JAR
  • create your own JAR
  • run your own JAR
  • Is your JAR executable? Are the properties files included?

Exercise

Test

  • commands are independent here, test them one by one
  • MissingException indicates a prepared problem, everything else is your mistake
  • just note which are failing, fix them later
  • in classroom exercises, testing and fixing will be interleaved

Fix

  • First time: get an overview of the code!
  • interpret the stack trace
  • Think! What needs to be done?
  • links to detailed instructions for both prepared problems

Build

  • fix, compile, build a new JAR
  • repeat until done
