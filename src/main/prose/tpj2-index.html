objectives:

  • set up the development environment

  • practice the steps for fixing a program

If you're an experienced Java developer, all of this will be familiar to you. But many students in my classes are not. This tutorial establishes a baseline of required tools and skills for the classroom exercises.

Prerequisites

To work through the exercises from the command line, you need a web browser, a text editor, and a Java Development Kit (JDK). If available for your platform, you can use the OpenJDK. Beware of the Oracle Java installers for Windows and OS X, they have a reputation for installing adware along with the Java browser plug-in.

Execute the following commands to verify that a JDK is available:

java -version
javac -version

If you prefer to code with an Integrated Development Environment (IDE), there are several that support Java development.
For example IntelliJ, Eclipse, Visual Studio Code, or NetBeans.

For convenience, bookmark the Java SE API documentation of the Java version you are using.
When working from the command line, collect frequently used commands in a text file for copy&paste.
