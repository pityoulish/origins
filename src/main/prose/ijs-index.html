Instructor's Notes: Message Board with Sockets

Objective of this exercise...

Overview... it's about the Message Board and Java Sockets. With a Binary Protocol.

What You Need

mbserver-sockets.jar
The server program. Usage instructions:
  java -jar mbserver-sockets.jar X
  • Tutorial. Give as homework, one or two weeks of headroom. Details to be fleshed out.
  • Server. Any Netbook with a recent JDK will do. Video output (HDMI, VGA?) to connect to projector or other shared display.
  • Network. Wireless access point without Internet connection will do. Might even be better than having an Internet connection: less distraction, no access to Pityoulish sources.
  • Software. Local web server with exercise and Java API docs; optional if Internet connection is available. Message Board socket server, see box.
  • Client. Different machine! Only for the rehearsal, but you MUST rehearse.

Rehearse

Tech Rehearsal.

  • Connect server to network. Look up IP address or hostname.
  • Start web server. Connect client to network. Access web server from client: Exercise, Java API docs. Use IP address or hostname in URL? http or https?
  • Start Message Board Socket server in terminal. Fixed port or random port? Access Message Board server from other machine. Use IP address or hostname? Firewall in the way?
  • Connect server to projector or other shared display. Do you get a picture? Need to adjust graphic adapter settings? Mirror display or extend desktop?
  • Move terminal to shared display. Adjust window size, 80 columns or more recommended. Output readable from the back row? Adjust font size and colors.
