Utility classes for implementing the binary TLV protocol. The protocol defines messages with TLV structures according to the Basic Encoding Rules (BER). All TLV types (aka tags) used in the protocol are defined in the {@link pityoulish.sockets.tlv.ProtocolConstants.TLVType TLVType} enum. The {@link pityoulish.sockets.tlv.AnyTLV} class represents a TLV structure in memory. Use the derived class {@link pityoulish.sockets.tlv.BuildTLV} to create a new TLV, or {@link pityoulish.sockets.tlv.ParseTLV} to interpret an obtained TLV. Have a look at HowToBuildTLV.java and HowToParseTLV.java, respectively, to learn how to use these classes.
